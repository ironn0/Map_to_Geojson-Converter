# Examples for Map to GeoJSON Converter

This folder contains sample inputs and outputs to test the converter.

## Sample Files
- `sample_map.png`: A simple map image of Italy (use for image-to-GeoJSON).
- `sample_output.geojson`: Expected GeoJSON output from the sample image.
- `sample_svg.svg`: An SVG map file (use for SVG-to-GeoJSON).

## How to Use
1. Run the script with the sample input.
2. Compare output with the provided GeoJSON.
3. Visualize at https://geojson.io.

Note: Sample images are placeholders; replace with real files for testing.
